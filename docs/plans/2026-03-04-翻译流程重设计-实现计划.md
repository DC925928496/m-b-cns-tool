# 翻译流程重设计（WPF + CLI + DLL 动态注入）Implementation Plan

> **For Claude:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task.

**Goal:** 按骑砍2本地化规范重做 WPF+CLI 翻译流程，新增“仅限 TextObject 构造字面量”的 DLL 扫描与运行时注入，生成依赖原 Mod 的外挂汉化包，并实现记录复用。

**Architecture:** 以“工程记录文件（Project Record）”作为扫描/翻译/校对/打包的唯一中间态；扫描阶段构建条目列表并回填历史记录与缓存；打包阶段写语言文件并在需要时附带注入子模块与 `runtime_localization.json`（数组 + Base64 键）；注入子模块运行时动态枚举 `.ctor(string, ...)` 并 patch，严格版本门禁与失败隔离。

**Tech Stack:** .NET 8、C#、WPF、xUnit、Mono.Cecil（DLL 扫描）、SQLite（现有缓存）、Harmony（运行时注入）、System.Text.Json、XDocument
